# SCT_WD_4 To Do Web App

## 🚀 Overview

The **SCT_WD_4 To Do Web App** is a highly polished, interactive, and fully functional task management dashboard. Designed with a premium "dark glassmorphism" aesthetic, this productivity tool seamlessly combines elegant UI with robust, pure Vanilla JavaScript logic. 

Whether you're managing daily chores, tracking work deadlines, or focusing on study sessions, this application provides an intuitive and visually impressive environment to keep you organized.

Experience the live application here:

## ✨ Key Features

- **Advanced Task Management**: Create, edit, delete, and mark tasks as complete with real-time DOM updates.
- **Dynamic Filtering & Search**: Instantly filter your tasks by categories (Work, Personal, Study, etc.), status (Pending/Completed), priority, or search by keyword.
- **Drag-and-Drop Reordering**: Smooth, animated drag-and-drop capability to prioritize tasks visually on the fly.
- **Focus Timer (Pomodoro)**: A built-in 25-minute Pomodoro timer widget to maintain productivity during intense work sessions.
- **Persistent Local Storage**: Never lose your tasks! All data, including your preferred theme, is saved securely in your browser's local storage.
- **Overdue Task Highlighting**: Tasks that pass their assigned due date and time automatically receive a glowing red indicator.
- **Stunning UI/UX**:
  - Animated, floating gradient background blobs.
  - Frosted glass cards (Glassmorphism).
  - Smooth staggered slide-up animations for task items.
  - Confetti celebration upon completing all tasks.
- **Light & Dark Mode**: Seamlessly toggle between dark and light themes with the click of a button.

## 🛠️ Tech Stack

This project was built entirely without the need for heavy frameworks or build tools, relying purely on the fundamentals of the web:

- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom variables, Grid/Flexbox layouts, `@keyframes` animations, and `backdrop-filter` for the glass effect.
- **JavaScript (Vanilla ES6+)**: DOM manipulation, event handling, Local Storage API, and complex state management.
- **Third-Party Integrations**:
  - [Lucide Icons](https://lucide.dev/) for crisp, scalable iconography.
  - [SortableJS](https://sortablejs.github.io/Sortable/) for buttery-smooth drag-and-drop interactions.
  - [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti) for celebratory animations.
  - **Google Fonts**: *Poppins* for headings and *Inter* for body text.

## 💡 How to Run Locally

Since this is a vanilla frontend application, setup is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/varung-coder/SCT_WD_4.git
   ```
2. **Open the project directory:**
   ```bash
   cd SCT_WD_4
   ```
3. **Launch the application:**
   Simply double-click the `index.html` file to open it in your default web browser, or use a tool like the **Live Server** extension in VS Code for a better development experience.

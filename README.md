# TaskFlow Pro — Smart To-Do & Productivity Manager

![TaskFlow Pro Banner](https://via.placeholder.com/1200x400/0f111a/8b5cf6?text=TaskFlow+Pro+-+Productivity+Manager)

TaskFlow Pro is a highly polished, modern, and feature-rich To-Do web application. Built for high performance and an exceptional user experience, it features a premium "dark glassmorphism" UI, smooth micro-interactions, advanced task tracking, and a built-in Pomodoro timer to maximize your productivity.

It was designed specifically as an internship-level portfolio project showcasing advanced frontend skills using Vanilla Web Technologies.

## 🌟 Features

- **Modern UI/UX**: Premium dark and light mode themes built using CSS variables, featuring smooth transitions and glassmorphism effects.
- **Advanced Task Management**: Add, edit, delete, and categorize your tasks with due dates, priorities, and descriptions.
- **Drag & Drop Reordering**: Effortlessly organize your task list with intuitive HTML5 drag-and-drop.
- **Real-Time Search & Filters**: Quickly find what you need by searching titles or filtering by categories, completion status, and priority.
- **Dashboard Statistics**: Keep track of your productivity percentage, completed tasks, and total pending tasks with dynamic animated progress bars.
- **Pomodoro Widget**: Built-in 25-minute Pomodoro focus timer.
- **Persistent Storage**: All tasks and preferences are saved locally in the browser (Local Storage API).
- **Celebration Mode**: Enjoy a burst of confetti when you complete all your pending tasks!
- **Fully Responsive**: Flawless experience across desktops, tablets, and mobile devices with an interactive sliding sidebar.
- **Toast Notifications**: Non-intrusive, elegant notifications for user actions.

## 🛠️ Technologies Used

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom properties (Variables), Flexbox, CSS Grid, animations (`@keyframes`), Backdrop-Filter for glassmorphism.
- **JavaScript (Vanilla)**: ES6+ syntax, DOM manipulation, LocalStorage API, HTML5 Drag & Drop API.
- **Lucide Icons**: For clean, modern SVG iconography.
- **Canvas Confetti**: A lightweight 3rd party library to handle the celebration animations.

## 🚀 Setup Instructions

Since this project is built entirely with Vanilla web technologies, no complex build steps or installations are required!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/taskflow-pro.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd taskflow-pro
   ```
3. **Run the project:**
   Simply open the `index.html` file in any modern web browser.
   - You can also use an extension like **Live Server** in VS Code for a better development experience with hot-reloading.
